## 소스다운로드

 1. [Clone or download] 버튼을 클릭하고 <Download Zip> 파일을 클릭합니다.  
 2. 안드로이드 스튜디오를 시작합니다. 그리고 <import project> 메뉴를 클릭합니다.
 3. 살펴볼 예제 폴더를 선택하고 <OK> 버튼을 클릭합니다
